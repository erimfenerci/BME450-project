# Title
Alzheimer's Disease Detection and Stage Classification using MRI Images

## Team members
Erim Fenerci (erimfenerci), Janina Li (janinalli34)

## Project description
This project aims to develop and train a convolutional neural network model to detect and classify the stages of Alzheimer’s disease using brain MRI images. We plan to use publicly available datasets, such as the Kaggle dataset named “Augmented Alzheimer MRI Dataset V2,” which includes MRI scans categorized as Non-Demented, Very Mild, Mild, and Moderate. We will evaluate the model using accuracy and loss metrics across training and validation sets. An image classification model that can accurately distinguish between these different stages of Alzheimer’s disease is very important for early diagnosis, as automated MRI analysis could help healthcare providers identify cognitive decline at an earlier stage and support timely intervention and treatment planning.
